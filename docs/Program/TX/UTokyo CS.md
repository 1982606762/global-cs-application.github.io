???+ abstract "项目优劣势" 

    Pros:
    
    * 优异的QS排名，国内不错的名声
    * 科研导向，有更多时间科研
    * 体验日本文化（如果你不喜欢，这就是一个扣分项）
    * 相对宽松的永居政策
    * (建议从花费，就业，读博，移民，人身安全等各方面展开)
    
    Cons:

    * 不会日语的话，在日本生活比较麻烦
    * 期待您的贡献！理性回答即可 :)
